Project 1 : Personal Portfolio Website

This is a personal portfolio website I created to showcase my skills as a web developer. I used a combination of HTML, CSS, Tailwind CSS, and JavaScript to build the site. It features a clean, modern design with responsive layout, smooth scrolling, and custom animations to create a polished user experience.

Key Features:

  • Responsive design using Tailwind CSS for fast and efficient styling.
  • Interactive elements, including smooth scroll and hover animations built with JavaScript.
  • Custom-designed layout that adapts to mobile, tablet, and desktop screens.
  • Contact form with validation to ensure user inputs are correct.

Technologies Used:

HTML, CSS, Tailwind CSS, JavaScript
